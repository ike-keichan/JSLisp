## Quick start
### Install
```
$ git clone https://github.com/ike-keichan/JSLisp.git
```

### Build & Launch
```
$ cd ./JSLisp
$ make test
```

### Example
#### example1
```
>> 1

>> -1.2

>> a

>> nil

```

#### example2
