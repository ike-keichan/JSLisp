## Quick start
### Install
```
$ git clone https://github.com/ike-keichan/JSLisp.git
```

### Setup & Launch
```
$ cd ./JSLisp
$ make test
```

### Example
#### example1
```
>> 1      // 1
```

#### example2
```
>> a      // a
```

#### example3
```
>> nil    // nil
```

#### example4
```
>> (+ 1 2) // (+ 1 2)
```

#### example5
```
>> '(1 . 2) // (quote (1 2))
```

#### example6
```
>> '(nil . 1) // (quote (nil . 1))
```

#### example6
```
>> '(1 . nil) // (quote (1 . NaN))
```
